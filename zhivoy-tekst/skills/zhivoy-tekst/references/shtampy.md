# Словарь штампов и замен

## Вводные-пустышки → убрать
- «Стоит отметить, что…»
- «Важно понимать, что…»
- «Не секрет, что…»
- «В современном мире / в наше время / на сегодняшний день»
- «Давайте разберёмся / поговорим о том»
- «Как известно»
- «Хочется отметить»

Начинай сразу с факта.

## Канцелярит → живой глагол
| Было | Стало |
|------|-------|
| осуществляет доставку | доставляет |
| производит оплату | платит |
| является решением | решает |
| данный продукт | этот продукт / он |
| в целях экономии | чтобы сэкономить |
| в рамках проекта | в проекте |
| посредством сервиса | через сервис |
| в связи с тем, что | потому что |
| по причине того что | из-за |
| на данный момент | сейчас |

## Усилители-паразиты → убрать
«действительно», «поистине», «по-настоящему», «крайне», «весьма»,
«невероятно», «максимально», «абсолютно», «целый ряд», «широкий спектр».

## Шаблонные обороты нейросетей → переписать
- «Это не просто X — это Y» → сказать Y напрямую
- «В мире, где…» → убрать зачин
- «Откройте для себя…» → конкретное действие
- «Погрузитесь в…» → что именно делает читатель
- «Раскройте свой потенциал» → конкретный результат
- «Решения, которые меняют правила игры» → что меняется
- «Бесшовная интеграция» → как работает на деле

## Триады → две вещи или одна точная
«Удобно, быстро и надёжно», «качественно, профессионально и в срок» —
ритм-маркер машины. Оставь самое сильное.

## Пассив → актив
- «было принято решение» → «мы решили»
- «будет рассмотрено» → «я рассмотрю» / «команда рассмотрит»
- «ошибка была допущена» → «я ошибся»
